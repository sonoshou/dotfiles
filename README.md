# Install

````
# Install dotfiles
cd ~
git clone --recursive https://github.com/sonoshou/dotfiles.git
./dotfiles/dotfilesLink.sh

# Install molokai
mkdir -p ~/.vim/colors/
wget https://raw.githubusercontent.com/tomasr/molokai/master/colors/molokai.vim
mv molokai.vim ~/.vim/colors/molokai.vim
````
